YummyKodik configuration

Оставляй https://api.yani.tv, если не используешь зеркало.
Путь на машине, где крутится Jellyfin, куда плагин будет писать структуру Series/Season 01/*.strm и *.nfo.
Внешний адрес Jellyfin, который должен быть доступен клиенту во время воспроизведения. Не указывай 127.0.0.1, если смотришь не с самого сервера.
Подстрока для приоритизации источников. Можно несколько через |.
Если включено, refresh будет создавать отдельные файлы на каждую озвучку, например S01E01 - AniLibria.strm. Внутри будет ссылка со своим tr=....
Включает подробные логи HTTP запросов/ответов к Kodik (url, form, bodySnippet). Может быть шумно — включай только для дебага.
Если включено, плагин будет подтягивать аниме из списка пользователя через API /users/{id}/lists/{list_id}. Для этого нужен user token.
Если указан, будет использован как Authorization: Bearer. Иначе плагин попробует логиниться по учётке ниже.
Пароль будет сохранён в конфиге Jellyfin. Если это не ок, используй только токен.
Оставь пустым, если не требуется.
Reload discards unsaved edits.
